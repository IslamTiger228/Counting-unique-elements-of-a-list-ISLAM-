# Counting-unique-elements-of-a-list-ISLAM-
Данная программа написана на языке Python и предназначена для подсчета количества уникальных элементов в списке.  Пользователь вводит элементы списка, после чего программа определяет, сколько различных (уникальных) значений содержится в этом списке.

# Как работает программа

Пользователь вводит элементы списка через пробел

Программа преобразует строку в список

Используется структура данных для хранения уникальных значений

Подсчитывается количество уникальных элементов

Результат выводится на экран

# Код программы

Ввод элементов списка : isl= input("Введите элементы списка через пробел: ")

Преобразование строки в список : w = isl.split()

Получение уникальных элементов с помощью множества : unique_w= set(w)

Подсчет количества уникальных элементов : count = len(unique_w)

Вывод/подсчет количества уникальных элементов : print("Количество уникальных элементов:", count)

# Пример работы
Введите элементы списка через пробел: 1 2 2 3 4 4 5

Количество уникальных элементов: 5
# Используемые функции

input() — ввод данных пользователем

split() — разделение строки на список

set() — создание множества (уникальные элементы)

len() — подсчет количества элементов
